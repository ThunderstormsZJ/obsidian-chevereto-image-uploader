# Obsidian Chevereto Image Uploader

This plugin could resize(optional) and upload the image in your clipboard to chevereto when pasting.

## Getting started

### Settings

1. Api Endpoint: the Endpoint of the image hosting api.example：https://your_domain/api/1/upload
2. Chevereto API Token: the API token of the chevereto. open url: https://your_domian/dashboard/settings/api to obtain
3. Enable Resize: whether resizing images before uploading.
4. Max Width: images that wider than this will be resized resized by the natural aspect ratio. example：4096
