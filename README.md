## 介绍

此插件能够将剪贴板的图片在粘贴到Obsidian时上传至chevereto，并支持调整图像大小。

## 新手入门

### 设置

1. Api Endpoint: Chevereto API链接
2. Chevereto API Token: Chevereto API令牌
3. Enable Resize: 是否开启上传前大小调整
4. Max Width: 调整最大宽度
